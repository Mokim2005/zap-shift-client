## 🚀 How to Use

### View the Changes

1. Start your dev server:
   ```bash
   npm run dev
   ```

2. Navigate to the homepage

3. Scroll through all sections to see animations

4. Try hovering over cards and icons

5. Test theme switching (if implemented)

### Customize Colors

Edit gradient colors in each component:

```javascript
// Find this in component files
gradient: "from-blue-500 to-cyan-500"

// Change to your brand colors
gradient: "from-pink-500 to-rose-500"
```

### Adjust Animations

Modify animation speeds:

```javascript
// Slower
transition={{ duration: 0.8 }}

// Faster
transition={{ duration: 0.3 }}
```

### Change Glass Opacity

```javascript
// Lighter
className="bg-white/40 dark:bg-white/5"

// Stronger
className="bg-white/80 dark:bg-white/15"
```
